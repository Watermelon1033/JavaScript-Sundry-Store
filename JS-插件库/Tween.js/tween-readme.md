## Content
1. 前端实现动画方式
    + (1) JavaScript
        - 定时器: `setTimeout` 和 `setInterval`
        - `requestAnimationFrame` (代表: Tween.js)
    + (2) CSS3
        - `transition` 和 `animation`
    + (2) HTML5
        - `Canvas`, `SVG`, `WebGL`

2. Tween.js 与 css 动画相比优劣:
    + (1) 优势
        - 更加灵活(链式补间...)
        - 可以定义多个动画，循环调用
        - 应用场景更广阔
    + (2) 弊端
        - 动画的的更新需要主动调用更新方法(依赖定时器或者动画主循环函数)
        - 性能没有 CSS 优
